# Windows portable build

Use this script to generate a distributable Windows package for Memorylux:

```powershell
powershell -ExecutionPolicy Bypass -File .\build-windows-portable.ps1
```

Output:

- `dist/windows-portable/`
- `dist/memorylux-windows-portable.zip`

The output is portable and does not require Flutter to be installed on the target machine. It still ships the app runtime files generated by Flutter, and on some Windows PCs the Microsoft Visual C++ runtime may be needed by the operating system.
